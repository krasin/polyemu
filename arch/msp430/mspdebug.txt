mspdebug - debugger / simulator for msp430.

Install:

sudo apt-get install mspdebug

Run simulator:

mspdebug sim

Simulator / debugger commands:

First step:

prog <path to msp430 elf file> -- this will init the simulator and load elf file into memory.
This is required to use the simulator, even if elf file is not needed.

regs - show registers
set R5 10 - set register value
md addr n - show memory contents
mw addr b0 b1 b2 ... - write memory contents
step - make a step.
